RSA - Açıq açarlı kriptosistem

Oxunub: Müəllif:

 

Ağac strukturları nizamlama və axtarış əməliyyatlarında ən çox istifadə edilən alqoritmlərdir. Ağac strukturları Data Structures kursunun əsas bölmərindən biridir. Kompüter elmlərində ən məşhur ağac strukturlarından biri İkilik Ağac (Binary Tree) strukturudur. Daha sürətli axtarış və nizamlama üçün İkilik axtarış ağacı (Binary Search Tree) və İkilik Yığın (Binary Heaps) alqoritmlərindən istifadə edilir. Qraflar nəzəriyyəsində İkilik Ağaclar istiqamətlənməmiş qraflardır. İkilik ağacların hər düyümündə (parent node) ən çox iki uşaq(child node) düyümü ola bilər. Bir və ya sıfır uşağı olan düyümlər mümkündür. Sıfır uşağı olan düyümlərə yarpaq (leaf) deyilir. Ən birinci düyüm isə kök (root) adlanır.

 

 

Yuxarıdakı ağacda A düyümü Kök, B və C düyümləri Valideyin düyümləri, D, E, F və C düyümləri isə yarpaqlardır. Kökdən yarpağa qədər düyünlərin sayına yüksəklik (height) və ya səviyyə (level) deyilir. B və C düyünlərin yüksəklikləri eyni, D, E, F və G yarpaqlarının da yüksəklikləri eynidir. İkilik ağacların aşağıdakı növləri vardır:
⦁ Full Binary Tree
⦁ Perfect Binary Tree
⦁ Complate Binary Tree
⦁ Balanced Binary Tree

Full Binary Tree hər düyümdə ya iki (2) yada sıfır (0) uşaq düyümü olan İkilik ağac növüdür. Başqa sözlə belə deyə bilərik ki Full Binary Tree strukturunda yarpaqlardan başqa bütün düyünlərin iki uşaq düyünü vardır. h hündürlükdə Full Binary Tree struktrlu ağacda n yarpaq varsa düyümlərin sayı: \(d = 2n-1 = 2h +1 \) düsturu ilə hesablanır. Bundan əlavə yarpaqlar qeyri yarpaqlardan hər zaman bir ədəd çox olur.

Perfect Binary Tree hər düyünündə iki uşaq düyünü olan və bütün yarpaqlarının eyni yüksəklikdə olduğu bir İkilik ağac növüdür. h yüksəklikli bir Perfect Binary Tree də düyümlərin sayı qədərdir. Yarpaqlar isə düsturu ilə hesablanır. Qeyri-yarpaqlar və ya valideyin düyünlər(internal node or parent node)

Complate Binary Tree modelində sonuncu səviyyə istisna olmaqla bütün səviyyələr doldurulmalıdır və düyümlər ağaca soldan saga doğru əlavə olunmalıdır.

Balanced Binary Tree her bir düyümün sol və sağ alt ağaclarının yüksəklik fərqinin maksimum bir (1) olduğu bir ağac strukturudur. Bu ağaclara missal olaraq
AVL Trees,
B-Trees/2-3-4 Trees,
BB[α] Trees,
Red-black Trees,
Splay-Trees,
Skip Lists,
Scapegoat Trees,
Treaps və s.. nümunə göstərmək olar.



Ağac üzərində gəzmək (Tree Traversal) Xətti (vektorial) data strukturlarının əksinə Ağac strukturları üzərində gəzmək üçün bir çox metodlar vardır. Bunlara baxaq.
⦁ Depth First Traversals:
  ⦁ Inorder : Sol -> Kök -> Sağ
  ⦁ Preorder : Kök -> Sol -> Sağ
  ⦁ Postorder : Sol -> Sağ -> Kök
⦁ Breadth First Traversals:
  ⦁ Level Order Traversal : Kök -> Soldan Sağa
Bu keçid əməliyyatlarını aşağıda verilmiş ağac üzərində daha ətraflı araşdıraq.




Mövzunun davamı olaraq Binary Search Tree (İkilik axtarış ağacları) və Binary Heap (İkilik yığın) alqoritmlərindən bəhs edəcəyik və nümunə kodlar veriləcək. Oxuduğunuz üçün təşəkkürlər.
